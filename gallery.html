A tool that allows user to import data via CSV file or manual entry and calcualte advanced statistic on the data sets. The user can also generate relvant graphs and figures relvant to imported data set. The user may also save report in the form of a text file and calculated statistics in the form of a CSV file. Slide Grabber - 2017

A program used to scrape the daily lecture slides hosted on a former professor's website. This script was set to run every Monday and Wednesday at the beginning of class time during the Spring 2017 semester. The scheduling of execution was done using Cron on a linux machine. This was written in Python 3 using BeautifulSoup library.
